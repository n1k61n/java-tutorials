
Massivlər (Arrays) hər bir dəyər üçün ayrı-ayrı dəyişənlər elan etmək əvəzinə, birdən çox dəyəri **tək bir dəyişəndə** saxlamaq üçün istifadə olunur.

Massiv elan etmək üçün, dəyişən tipini **kvadrat mötərizələrlə** `[ ]` müəyyənləşdirin:

Java

```
String[] cars;
```

Biz indi sətirlər (strings) massivini saxlayan bir dəyişən elan etmişik. Ona dəyərlər daxil etmək üçün, dəyərləri **vergüllə ayrılmış siyahıda**, **fiqurlu mötərizələrin** `{ }` daxilində yerləşdirə bilərsiniz:

Java

```
String[] cars = {"Volvo", "BMW", "Ford", "Mazda"};
```

Tam ədədlərdən (integers) ibarət bir massiv yaratmaq üçün isə belə yaza bilərsiniz:

Java

```
int[] myNum = {10, 20, 30, 40};
```
