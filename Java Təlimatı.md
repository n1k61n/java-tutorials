
Java — geniş istifadə olunan obyekt yönümlü proqramlaşdırma dilidir. Mobil tətbiqlər, veb saytlar, masaüstü proqramlar və oyunlar yaratmaq üçün istifadə olunur.

###  Təlimatın bölmələri

| Bölmə                             | Məzmun                                                                                      |
| --------------------------------- | ------------------------------------------------------------------------------------------- |
| **Java Sintaksisi**               | Kodun yazılış qaydaları, əsas strukturlar                                                   |
| **Dəyişənlər və Məlumat Növləri** | `int`, `double`, `String`, `boolean` və s.                                                  |
| **Operatorlar və Şərtlər**        | `+`, `-`, `*`, `if`, `else`, `switch`                                                       |
| **Döngülər**                      | `for`, `while`, `do-while`, `break`, `continue`                                             |
| **Massivlər (Arrays)**            | Tək və çoxölçülü massivlərlə işləmək                                                        |
| **Metodlar**                      | Metodların yaradılması, parametrlər, aşırı yükləmə                                          |
| **Siniflər və Obyektlər**         | OOP əsasları: atributlar, metodlar, konstruktorlar                                          |
| **İrsi və Polimorfizm**           | `extends`, `super`, metodların yenidən yazılması                                            |
| **Abstraksiya və İnterfeys**      | `abstract`, `interface`, anonim siniflər                                                    |
| **Fayllarla İş**                  | Fayl yaratmaq, oxumaq, yazmaq, silmək                                                       |
| **I/O Axınları**                  | `FileInputStream`, `BufferedReader`, `BufferedWriter`                                       |
| **Kolleksiyalar və Strukturlar**  | `List`, `Set`, `Map`, `Iterator`, `ArrayList`, `HashMap`                                    |
| **İstisnalar və Səhvlər**         | `try`, `catch`, `finally`, `throw`, `throws`                                                |
| **İrəliləmiş Mövzular**           | `Generics`, `Annotations`, `RegEx`, `Threads`, `Lambda`                                     |
| **Layihə nümunələri**             | Sadə alqoritmlər: ədədlərin çevrilməsi, massivlərin sıralanması, palindrome yoxlaması və s. |


###  İstinadlar

| İstinad                           | Təsvir                                         |
| --------------------------------- | ---------------------------------------------- |
| **Java Açar Sözlər**              | `public`, `static`, `void`, `class` və s.      |
| **String və Math Metodları**      | `length()`, `substring()`, `Math.sqrt()` və s. |
| **Array və Collection Metodları** | `add()`, `remove()`, `sort()`, `contains()`    |
| **Fayl və Scanner Metodları**     | `nextLine()`, `read()`, `write()`              |
| **Sistem və Iterator Metodları**  | `System.out.println()`, `hasNext()`, `next()`  |
